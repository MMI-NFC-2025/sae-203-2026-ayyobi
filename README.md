# Festival Astro

Structure Astro pour un site de festival.

## Arborescence principale

- `BACKEND/`
- `public/`
- `src/components/`
- `src/layouts/`
- `src/lib/`
- `src/pages/`
- `src/styles/`

## Demarrage

```bash
npm run dev
```

## Variable d'environnement

Copier `.env.example` vers `.env` puis ajuster `PUBLIC_POCKETBASE_URL` si necessaire.
